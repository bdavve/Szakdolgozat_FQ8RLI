# Piacvezető nagy nyelvi modellek összehasonlítása a szoftverfejlesztés támogatásában imperatív és deklaratív nyelvek esetén (C#, T-SQL)

**Szakdolgozat — Budapesti Corvinus Egyetem, 2026**

Berki Dávid

Témavezető: Magyar Melinda

A dolgozat három piacvezető nagy nyelvi modell (ChatGPT GPT-5.4 Thinking, Claude Opus 4.6, Gemini 3.1 Pro) kódgenerálási képességeit hasonlítja össze imperatív (C#) és deklaratív (T-SQL) programozási feladatokon keresztül, öt kódminőségi dimenzió mentén.
